Creación de un Proyecto Angular

Iniciando un Nuevo Proyecto

Con Angular CLI instalado, crear un nuevo proyecto Angular es un proceso sencillo. Sigue estos pasos para iniciar tu primer proyecto:

Comando para Crear un Nuevo Proyecto

    ng new mi-primera-app

Este comando generará una nueva carpeta llamada mi-primera-app con toda la estructura y archivos necesarios para empezar.

Configuración Inicial

Durante la ejecución del comando, Angular CLI te pedirá que respondas algunas preguntas para configurar tu proyecto:

  • ¿Agregar Angular Routing? Escribe y si planeas utilizar rutas en tu aplicación.
  • Tipo de Hoja de Estilos: Selecciona el preprocesador CSS que prefieras (CSS, SCSS, Sass, Less, Stylus). Para este tutorial, selecciona CSS.

Explorando la Estructura del Proyecto

Después de crear el proyecto, navega al directorio del mismo:

    cd mi-primera-app

Dentro de este directorio, encontrarás la siguiente estructura:

  • e2e/: Contiene pruebas end-to-end.
  • node_modules/: Carpeta de dependencias instaladas.
  • src/: Código fuente de la aplicación.
  • angular.json: Configuración del proyecto Angular.
  • package.json: Información del proyecto y dependencias.
  • tsconfig.json: Configuración de TypeScript.

Ejecutando la Aplicación

Para ver tu aplicación en funcionamiento, ejecuta el siguiente comando:

    ng serve

Este comando compila la aplicación y la sirve en un servidor de desarrollo. Por defecto, estará disponible en http://localhost:4200.

Accediendo a la Aplicación en el Navegador

Abre tu navegador web favorito y visita http://localhost:4200. Deberías ver una página de bienvenida de Angular indicando que tu aplicación está corriendo correctamente.

Detalles del Comando ng serve

  • --open (o -o): Abre automáticamente la aplicación en el navegador después de compilar.
  • --port: Especifica un puerto diferente al 4200.
  • --watch: Vuelve a compilar la aplicación cuando se detectan cambios en los archivos.
