Primer Componente en Angular

Los componentes son la piedra angular de las aplicaciones Angular. Cada componente controla una parte específica de la interfaz de usuario y su comportamiento asociado.

Generando un Nuevo Componente

Angular CLI facilita la creación de nuevos componentes mediante comandos simples. Para generar un nuevo componente llamado hola-mundo, ejecuta el siguiente comando en la terminal:

    ng generate component hola-mundo

Este comando creará una carpeta llamada hola-mundo dentro de src/app/ con los siguientes archivos:

  • hola-mundo.component.ts: Contiene la lógica del componente.
  • hola-mundo.component.html: Plantilla HTML del componente.
  • hola-mundo.component.css: Estilos específicos para el componente.
  • hola-mundo.component.spec.ts: Archivo de pruebas unitarias.

Modificando la Plantilla del Componente

Vamos a personalizar la plantilla del componente para mostrar un mensaje de bienvenida.

Archivo: hola-mundo.component.html

    <h2>¡Hola Mundo!</h2>
    <p>Este es mi primer componente en Angular.</p>

Incluyendo el Componente en la Aplicación

Para que el nuevo componente se muestre en la aplicación, debemos incluir su selector en la plantilla del componente raíz.

Archivo: app.component.html

    <app-hola-mundo></app-hola-mundo>

Después de guardar los cambios, verifica que tu servidor de desarrollo esté corriendo (ng serve) y refresca la página en el navegador. Deberías ver el mensaje de bienvenida de tu nuevo componente.
