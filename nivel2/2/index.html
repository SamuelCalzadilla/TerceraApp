Rutas en Angular

El sistema de enrutamiento en Angular permite navegar entre diferentes vistas o componentes dentro de una aplicación de una sola página (SPA). Las rutas definen cómo se carga y se muestra el contenido en función de la URL.

Configurando el Enrutador

Para configurar el enrutador en tu proyecto Angular, sigue estos pasos:

1. Importar Módulos Necesarios

En app.module.ts, importa RouterModule y Routes desde @angular/router:

        import { RouterModule, Routes } from '@angular/router';
import { HolaMundoComponent } from './hola-mundo/hola-mundo.component';
import { OtroComponenteComponent } from './otro-componente/otro-componente.component';

const routes: Routes = [
  { path: '', component: HolaMundoComponent },
  { path: 'otro', component: OtroComponenteComponent }
];

@NgModule({
  declarations: [
    AppComponent,
    HolaMundoComponent,
    OtroComponenteComponent
  ],
  imports: [
    BrowserModule,
    FormsModule,
    RouterModule.forRoot(routes)
  ],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule { }

En este ejemplo, se definen dos rutas:

  • Ruta Vacía (''): Carga el componente HolaMundoComponent.
  • Ruta 'otro': Carga el componente OtroComponenteComponent.

2. Crear el Componente de Ruta

Genera un nuevo componente llamado otro-componente:

ng generate component otro-componente

En otro-componente.component.html, añade contenido:

        <h2>Otro Componente</h2>
<p>Este es otro componente de la aplicación.</p>

3. Añadir Enlaces de Navegación

Para permitir la navegación entre rutas, actualiza la plantilla app.component.html con enlaces de navegación y un router-outlet:

        <nav>
  <a routerLink="" routerLinkActive="active">Inicio</a> |
  <a routerLink="/otro" routerLinkActive="active">Otro</a>
</nav>
<router-outlet></router-outlet>

El elemento router-outlet actúa como un marcador de posición donde se renderizan los componentes de las rutas correspondientes.

4. Agregar Estilos a la Navegación

Para mejorar la apariencia de la navegación, puedes agregar estilos en styles.css:

        nav a {
    margin-right: 15px;
    text-decoration: none;
    color: #2980b9;
}

nav a.active {
    font-weight: bold;
    color: #2c3e50;
}

Esto resaltará el enlace activo y mejorará la experiencia de navegación.

Protección de Rutas con Guardias

Angular permite proteger rutas mediante guardias, que son servicios que determinan si se puede acceder a una ruta específica.

Creando un Guard para Autenticación

Genera un guard llamado auth:

ng generate guard auth

Implementa la lógica de autenticación en el guard:

        import { Injectable } from '@angular/core';
import { CanActivate, ActivatedRouteSnapshot, RouterStateSnapshot, UrlTree, Router } from '@angular/router';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class AuthGuard implements CanActivate {

  constructor(private router: Router) { }

  canActivate(
    next: ActivatedRouteSnapshot,
    state: RouterStateSnapshot): boolean {
      const estaAutenticado = false; // Reemplaza con lógica real de autenticación
      if (estaAutenticado) {
        return true;
      } else {
        this.router.navigate(['/']);
        return false;
      }
  }
  
}

Actualiza la ruta que deseas proteger agregando el guard:

        const routes: Routes = [
  { path: '', component: HolaMundoComponent },
  { path: 'otro', component: OtroComponenteComponent, canActivate: [AuthGuard] }
];

Ahora, solo los usuarios autenticados podrán acceder a la ruta /otro.

Navegación Programática

Además de los enlaces de navegación, puedes navegar programáticamente a través de código en tus componentes utilizando el servicio Router.

Ejemplo de Navegación Programática

En hola-mundo.component.ts, inyecta Router y crea un método para navegar:

        import { Router } from '@angular/router';

constructor(private dataService: DataService, private router: Router) { }

navegarAOtro(): void {
    this.router.navigate(['/otro']);
}

En la plantilla hola-mundo.component.html, añade un botón para activar la navegación:

<button (click)="navegarAOtro()">Ir a Otro Componente</button>

Al hacer clic en el botón, la aplicación navegará a la ruta /otro.
