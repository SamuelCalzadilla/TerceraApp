Directivas en Angular

Las directivas en Angular son clases que permiten manipular el DOM de formas específicas. Son fundamentales para extender la funcionalidad del HTML y crear componentes más dinámicos y reutilizables.

Tipos de Directivas en Angular

  • Directivas Estructurales: Alteran la estructura del DOM, añadiendo o eliminando elementos. Ejemplos: *ngIf, *ngFor.
  • Directivas de Atributo: Cambian la apariencia o el comportamiento de un elemento existente. Ejemplos: ngStyle, ngClass.
  • Directivas Personalizadas: Creadas por el desarrollador para añadir funcionalidades específicas.

Directivas Estructurales

1. *ngIf

Controla la renderización de un elemento basado en una condición booleana.

Ejemplo de *ngIf

En hola-mundo.component.ts, añade una propiedad booleana:

mostrarSeccion: boolean = true;

En la plantilla hola-mundo.component.html, usa *ngIf:

        <div *ngIf="mostrarSeccion">
  <p>Esta sección se muestra si mostrarSeccion es true.</p>
</div>
<button (click)="mostrarSeccion = !mostrarSeccion">Toggle Sección</button>

Al hacer clic en el botón, la sección se alternará entre mostrar y ocultar.

2. *ngFor

Itera sobre una colección y renderiza un elemento por cada ítem.

Ejemplo de *ngFor

En hola-mundo.component.ts, define una lista de usuarios:

usuarios: string[] = ["Ana", "Luis", "Carlos", "Marta"];

En la plantilla hola-mundo.component.html, usa *ngFor para listar los usuarios:

        <ul>
  <li *ngFor="let usuario of usuarios">{{ usuario }}</li>
</ul>

Directivas de Atributo

1. ngStyle

Permite aplicar estilos en línea dinámicamente a un elemento.

Ejemplo de ngStyle

En hola-mundo.component.ts, define un objeto de estilos:

        estilos: any = {
  color: 'blue',
  fontSize: '20px'
};

En la plantilla hola-mundo.component.html, aplica los estilos:

<p [ngStyle]="estilos">Este párrafo tiene estilos dinámicos.</p>

2. ngClass

Permite agregar o quitar clases CSS dinámicamente a un elemento.

Ejemplo de ngClass

Define una clase CSS en styles.css:

        .destacado {
    background-color: #f1c40f;
    padding: 10px;
}

En hola-mundo.component.ts, define una propiedad booleana:

esDestacado: boolean = true;

En la plantilla hola-mundo.component.html, usa ngClass:

        <p [ngClass]="{'destacado': esDestacado}">Este párrafo puede ser destacado.</p>
<button (click)="esDestacado = !esDestacado">Toggle Destacado</button>

Al hacer clic en el botón, se alternará la clase destacado, cambiando el estilo del párrafo.

Directivas Personalizadas

Angular permite crear directivas personalizadas para encapsular comportamientos específicos y reutilizables.

Creando una Directiva Personalizada

Vamos a crear una directiva que cambia el color de fondo de un elemento al pasar el cursor sobre él.

Generando la Directiva

ng generate directive resaltar

Implementando la Directiva

En resaltar.directive.ts, implementa la lógica para cambiar el color de fondo:

        import { Directive, ElementRef, HostListener, Input } from '@angular/core';

@Directive({
  selector: '[appResaltar]'
})
export class ResaltarDirective {
  @Input() appResaltar: string = '';

  constructor(private el: ElementRef) { }

  @HostListener('mouseenter') onMouseEnter() {
    this.resaltar(this.appResaltar || 'yellow');
  }

  @HostListener('mouseleave') onMouseLeave() {
    this.resaltar('');
  }

  private resaltar(color: string) {
    this.el.nativeElement.style.backgroundColor = color;
  }
}

Usando la Directiva en la Plantilla

En hola-mundo.component.html, aplica la directiva a un elemento:

<p [appResaltar]="'lightgreen'">Pasa el cursor sobre este párrafo para resaltar.</p>

Al pasar el cursor sobre el párrafo, el fondo cambiará a lightgreen, y volverá al estado original al retirar el cursor.

Práctica Recomendada

Crear directivas personalizadas para encapsular comportamientos repetitivos o específicos ayuda a mantener el código limpio y facilita la reutilización de componentes en diferentes partes de la aplicación.
