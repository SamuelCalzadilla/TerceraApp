Instalación de Angular

Requisitos Previos

Antes de comenzar con la instalación de Angular, asegúrate de tener instalados los siguientes componentes en tu sistema:

  • Node.js y npm: Angular requiere Node.js para su entorno de ejecución y npm como gestor de paquetes.
  • Editor de Código: Se recomienda utilizar editores como Visual Studio Code, Sublime Text o Atom.
  • Navegador Web: Un navegador moderno como Google Chrome, Firefox o Edge para probar las aplicaciones.

Instalando Node.js y npm

Node.js es una plataforma de JavaScript que permite ejecutar código fuera del navegador. npm (Node Package Manager) es el gestor de paquetes que viene con Node.js y es esencial para instalar Angular CLI y otras dependencias.

Pasos para Instalar Node.js y npm

  1. Visita la página oficial de Node.js: https://nodejs.org/
  2. Descarga el instalador recomendado para tu sistema operativo (Windows, macOS, Linux).
  3. Ejecuta el instalador y sigue las instrucciones en pantalla.
  4. Verifica la instalación abriendo una terminal o símbolo del sistema y ejecutando los siguientes comandos:
    node -v
    npm -v

Deberías ver las versiones instaladas de Node.js y npm, respectivamente.

Instalando Angular CLI

Angular CLI es una herramienta de línea de comandos que facilita la creación, desarrollo y gestión de proyectos Angular.

Pasos para Instalar Angular CLI

  1. Abre una terminal o símbolo del sistema.
  2. Ejecuta el siguiente comando para instalar Angular CLI globalmente:
        npm install -g @angular/cli

La opción -g indica que el paquete se instalará globalmente, permitiendo su uso en cualquier directorio.

Verificando la Instalación

Después de la instalación, verifica que Angular CLI se haya instalado correctamente ejecutando:

        ng version

Deberías ver información sobre la versión de Angular CLI y otros paquetes relacionados.

Solución de Problemas Comunes

  • Permisos Denegados: Si encuentras errores relacionados con permisos al instalar paquetes globales, considera ejecutar el comando con sudo (en macOS/Linux) o verifica los permisos de tu terminal.
  • Versiones Incompatibles: Asegúrate de tener versiones compatibles de Node.js y npm. Angular CLI generalmente soporta las versiones LTS (Long Term Support) de Node.js.
  • Problemas de Conexión: Si tienes problemas de conexión al descargar paquetes, verifica tu conexión a internet o intenta cambiar de red.

Actualizando Angular CLI

Para mantener Angular CLI actualizado a la última versión, ejecuta:

        npm install -g @angular/cli@latest

Esto actualizará Angular CLI globalmente a la versión más reciente disponible.
