Primer Componente en Angular

Los componentes son la piedra angular de las aplicaciones Angular. Cada componente controla una parte específica de la interfaz de usuario y su comportamiento asociado.

Generando un Nuevo Componente

Angular CLI facilita la creación de nuevos componentes mediante comandos simples. Para generar un nuevo componente llamado hola-mundo, ejecuta el siguiente comando en la terminal:

    ng generate component hola-mundo

Este comando creará una carpeta llamada hola-mundo dentro de src/app/ con los siguientes archivos:

  • hola-mundo.component.ts: Contiene la lógica del componente.
  • hola-mundo.component.html: Plantilla HTML del componente.
  • hola-mundo.component.css: Estilos específicos para el componente.
  • hola-mundo.component.spec.ts: Archivo de pruebas unitarias.

Estructura de un Componente

Un componente en Angular está compuesto por:

  • Decorador @Component: Define metadatos para el componente, como su selector, plantilla y estilos.
  • Clase del Componente: Contiene la lógica y los datos que se utilizan en la plantilla.

Ejemplo de hola-mundo.component.ts

    import { Component, OnInit } from '@angular/core';

@Component({
selector: 'app-hola-mundo',
templateUrl: './hola-mundo.component.html',
styleUrls: ['./hola-mundo.component.css']
})
export class HolaMundoComponent implements OnInit {

constructor() { }

ngOnInit(): void {
}

}

Modificando la Plantilla del Componente

Vamos a personalizar la plantilla del componente para mostrar un mensaje de bienvenida.

Archivo: hola-mundo.component.html

    <h2>¡Hola Mundo!</h2>
    <p>Este es mi primer componente en Angular.</p>

Incluyendo el Componente en la Aplicación

Para que el nuevo componente se muestre en la aplicación, debemos incluir su selector en la plantilla del componente raíz.

Archivo: app.component.html

    <app-hola-mundo></app-hola-mundo>

Después de guardar los cambios, verifica que tu servidor de desarrollo esté corriendo (ng serve) y refresca la página en el navegador. Deberías ver el mensaje de bienvenida de tu nuevo componente.

Personalizando el Componente

Ahora, vamos a agregar más funcionalidad y estilo al componente hola-mundo.

Agregando Propiedades y Métodos

En hola-mundo.component.ts, agrega una propiedad llamada mensaje y un método para actualizarla:

    export class HolaMundoComponent implements OnInit {
mensaje: string = "¡Hola Mundo! Este es mi primer componente.";

constructor() { }

ngOnInit(): void {
}

actualizarMensaje(nuevoMensaje: string): void {
this.mensaje = nuevoMensaje;
}
}

Actualizando la Plantilla para Mostrar la Propiedad

En hola-mundo.component.html, utiliza la interpolación para mostrar la propiedad mensaje y agrega un botón para actualizarlo:

    <h2>{{ mensaje }}</h2>
<button (click)="actualizarMensaje('Has actualizado el mensaje!')">Actualizar Mensaje</button>;

Guarda los archivos y observa cómo el mensaje se actualiza al hacer clic en el botón.

Estilizando el Componente

Para mejorar la apariencia del componente, agrega algunos estilos en hola-mundo.component.css:

    h2 {
color: #27ae60;
}

button {
margin-top: 10px;
}

Estos estilos cambiarán el color del título y agregarán un margen al botón.
